videojs-dashjs demo

(note: fullscreen is blocked in the jsfiddle iframe. See this version for fullscreen.)

This demo shows MPEG-DASH playback using Video.js as the player and Dash.js as the media controller (or "playback tech"). There is also a video.js plugin with this code.

For video.js, this adds support for MPEG-DASH in browsers that support the Media Source Extensions (currently IE11 and the latest version of Chrome).

For dash.js, this gives access to all of the video.js plugins, custom skinnning, and support for falling back to other formats/technologies (HLS/MP4/Webm, HTML5/Flash/Youtube).
Examples of custom video.js players:

  • Allure (Conde Nast)
  • Funny or Die
  • The Guardian

If you have any questions or comments, use the videojs-dashjs issue tracker.

DASH (through Media Source Extensions) is not supported in your browser, you are currently viewing a fallback MP4 playing through Flash or HTML5 video. Try using the latest version of IE or Chrome.
